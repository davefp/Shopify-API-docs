Hello, Shopify API!

Shopify’s API lets you programmatically do almost anything merchants can do with their Shopify stores. You can look over orders, modify products, research customers, play with collections, and more. In a nutshell:

Shopify API == OAuth 2 + HTTP + (JSON | XML) // REST

Adapters for Ruby, Python, PHP, Java, and Node.js exist, but all you really need is HTTP and some love. Give us a shout if you run into any trouble.
