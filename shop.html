What can you do with the shop object using the Shopify API?

The Shopify API lets you do the following with shops:

  • Get information about the shop.

The API doesn't let you do anything other than retrieve information about the shop. Only the shopowner can modify the information about the shop, and only manually via the General settings page in the admin.

Shop properties

Name Description
address1 The shop's street address.
city The city in which the shop is located.
country The country in which the shop is located.
created_at The date and time when the shop was created. The API returns this value in ISO 8601 format.
currency The three-letter code for the currency that the shop accepts.
domain The shop's domain.
email The contact email address for the shop.
id A unique numeric identifier for the shop.
money_format A string representing the way currency is formatted when the currency isn't specified.
money_with_currency_format A string representing the way currency is formatted when the currency is specified.
myshopify_domain The shop's "myshopify.com" domain.
name The name of the shop.
phone The contact phone number for the shop.
plan_name The name of the shop's pricing plan.
province The province or state of the shop's address.
public
shop_owner The username of the shopowner.
source
tax_shipping
taxes_included
timezone The name of the shop's time zone.
zip The zip or postal code of the shop's address.

Endpoints

GET /admin/shop.json

Example: Get the shop's properties

Request:

GET /admin/shop.json
