What can you do with product images using the Shopify API?

The Shopify API lets you do the following with product images:

  • Get information about one or more product images.
  • Add an image to a product.
  • Delete a product image.

Just about anything that can be done with an product image through the shop's admin page can also be done through the API.

Product image properties

Name Description
created_at The date and time when the product image was created. The API returns this value in ISO 8601 format.
id A unique numeric identifier for the product image.
position The order of the image in the list. The first image is at position 1 and is the "main" image for the product.
product-id The id of the product associated with the image.
src The URL of the image.
updated_at The date and time when the product image was last modified. The API returns this value in ISO 8601 format.

Endpoints

GET /admin/products/:product_id/images

Query parameters

By default, requesting a list of product images returns all the product images for that shop. You can use the optional query parameters listed below to filter the returned list by specific criteria.

Name Description
fields A comma-separated list of the properties you want returned for each product image in the result list. By default, the returned list of product images is a list of product images with all the properties included. Use this parameter to restrict the returned list of product images to showing only those properties you specify, with each property name separated by a comma.
since_id The lower bound on a product imageâ€™s id. Use this parameter to restrict the returned list to only product images whose id is greater than the specified id.

Example: Get the complete list of product images for a product

In this example, we want to retrieve the complete list of images for the product whose id is 632910392.

Request:

GET /admin/products/632910392/images.json

Response:

HTTP/1.1 200 OK[]
{
  "images": [
    {
      "created_at": "2012-03-13T16:09:58-04:00",
      "id": 850703190,
      "position": 1,
      "product_id": 632910392,
      "updated_at": "2012-03-13T16:09:58-04:00",
      "src": "http://static.shopify.com/s/files/1/6909/3384/products/ipod-nano.png?0"
    }
  ]
}
GET /admin/products/:product_id/images/:product_image_id.json

Example: Get a single product image

In this example, we want to retrieve an image for the product whose id is 632910392; the product image id is 850703190.

Request:

GET /admin/products/632910392/images/850703190.json

Response:

HTTP/1.1 200 OK[]

POST /admin/products/:product_id/images.json

Example: Add an image

Request:

GET admin/shops/241253187/product images.json

Response:
POST /admin/product image.json

Example: Add a base64-encoded image to a product

If you have an image in memory or local storage (e.g. your hard drive), you can upload it to Shopify by converting it into a string using base64 encoding and including it as an attachment property in the request body.

When you upload an image, you need to provide it with a filename.

To make a product image the primary one, set its position property to 1.

In this example:

  • We'll add a new product image to a product as a string created using base64 encoding. Since these strings are very long, we will use the string === BASE64-ENCODED IMAGE === to represent the image data.
  • The id of the product is 632910392.
  • We'll assign the filename rails_logo.gif to the product image.

Request:

POST /admin/products/:product_id/images.json

Request body:

{
  "image": {
    "position": 1,
    "attachment": "=== BASE64-ENCODED IMAGE ==="
    "filename": "rails_logo.gif"
  }
}

Response:

HTTP/1.1 201 Created[]
{
  "image": {
    "created_at": "2012-03-13T16:10:43-04:00",
    "id": 916933559,
    "position": 1,
    "product_id": 632910392,
    "updated_at": "2012-03-13T16:10:43-04:00",
    "src": "http://static.shopify.com/s/files/1/6909/3384/products/rails_logo.gif?1"
  }
}

Example: Add an image from an URL to a product

In this example, we'll instruct Shopify to create its own copy of the image stored at http://example.com/rails_logo.gif and add it as a product image for the product whose id is 632910392.

Request:

POST /admin/products/632910392/images.json

Request body:

{
  "image": {
    "src": "http://example.com/rails_logo.gif"
  }
}

Response:

HTTP/1.1 201 Created[]
{
  "image": {
    "created_at": "2012-03-13T16:10:43-04:00",
    "id": 916933559,
    "position": 1,
    "product_id": 632910392,
    "updated_at": "2012-03-13T16:10:43-04:00",
    "src": "http://static.shopify.com/s/files/1/6909/3384/products/rails_logo.gif?1"
  }
}
