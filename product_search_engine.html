What can you do with product search engines using the Shopify API?

The Shopify API lets you do the following with product search engines:

  • Get information about the shop's product search engines

Product search engine properties

Name Description
created_at The date and time when the product search engine was created. The API returns this value in ISO 8601 format.
name The name of the product search engine.

Endpoints

GET /admin/product_search_engines.json

Query parameters

By default, requesting a list of product search engines returns all the product search engines for that shop. You can use the optional query parameters listed below to filter the returned list by specific criteria.

Name Description
fields A comma-separated list of the properties you want returned for each product search engine in the result list. By default, the returned list of product search engines is a list of product search engines with all the properties included. Use this parameter to restrict the returned list of product search engines to showing only those properties you specify, with each property name separated by a comma.

Example: Get a list of the shop's product search engines

Request:

GET admin/shops/241253187/product search engines.json

Response:

HTTP/1.1 200 OK[]
{
  "product_search_engines": [
    {
      "created_at": "2012-05-01T14:22:26-04:00",
      "name": "Google Product Search"
    }
  ]
}
